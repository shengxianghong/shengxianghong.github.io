**项目介绍:**
分享街平台区别于传统的O2O电商平台，融入分享经济概念，将会员的身份由单一的消费者转变成为优质商家的推广员，升级成为消费商，不但帮助商家更好的进行推广，更是拉近了商家、消费者、平台之间的关系，变单一的买卖服务关系为相互合作关系。消费者不但可以以实惠的价格购买到商品与服务，而且通过简单的分享与推荐就可以赚取分享红包收入以及推荐奖励收入，由一个普通的消费者升级成为消费商，获得更多额外的收益。平台帮助商家提供免费的在线展示与交易服务，并且通过会员消费后的朋友圈分享帮助商家引流扩大业绩，同时商家也可以主动推荐进店消费者成为分享街会员，形成推荐绑定关系，额外赚取所推荐会员在分享街所有加盟商家每笔消费的推荐奖励，获得巨额跨界收入。

**责任描述:**

**项目技术:**

3.集成了比较实用的第三方提供的功能 有：**极光推送 百度地图(包括定位、POI检索、导航等等) 友盟的分享、第三方登录 融云的即时聊天、微信支付宝支付等等**

其他方面：感觉客户没有对自己的App有准确的定位、前前后后加加减减使得App失去了最初的意义、显得过于繁琐、虽然初衷是好的、上线后搞的活动力度也大、用户也有好几万、但是一旦失去金钱的支持用户很快就会流失。
